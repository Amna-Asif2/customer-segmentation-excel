# Customer Segmentation Using Excel 

## Objective:
Segment customers based on demographics and purchasing behavior to identify high-value customers.

## Tools Used:
Excel (formulas, pivot tables, charts)

## Steps Taken:

### Data Cleaning
-Removed unnecessary columns 
-Handled missing values by deleting blank rows

### Feature Engineering

- Calculated `Customer_Age` from `Year_Birth`  
- Calculated `Total_Spending` across purchase categories (e.g., wines, fruits, meat)  
- Created `Customer_Segment` (High Value / Low Value) 

### Segmentation & Analysis

- Grouped customers by age (`Under 30`, `30â€“49`, `50+`)  
- Created pivot tables to find:
  - Average spending by age group  
  - Sum spending by marital status  
- Visualized insights using bar charts and pie charts

- Customers aged **50+** have the highest average spending  
- **High-value customers** represent a larger group  
- Married customers spend more on average than single customers
