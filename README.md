Beginner-friendly customer segmentation project using Excel to analyze purchasing behavior.

# Customer Segmentation Using Excel 

## Objective:
Segment customers based on demographics and purchasing behavior to identify high-value customers.

## Tools Used:
Excel (formulas, pivot tables, charts)

## Steps Taken:

## Data Cleaning
Removed unnecessary columns 
Handled missing values by deleting blank rows

## Feature Engineering

Calculated Customer_Age from Year_Birth
Calculated Total_Spending across purchase categories
Created Customer_Segment (High Value / Low Value)

## Segmentation & Analysis

Grouped customers by age (Under 30, 30–49, 50+)
Created pivot tables to find average spending by age and marital status
Visualized insights using bar charts and pie charts

## Business Insights

Customers aged 30–49 spend the most
High-value customers contribute most revenue
Married customers spend more on average than single customers

## Deliverables:

Excel workbook with clean data, formulas, pivot tables, and charts

## Outcome:

Gained hands-on Excel analytics experience
Identified high-value customer segments
